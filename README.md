# SSLSocket
An SSL socket server and client in JAVA

##Installation
```
git clone 
```

##Usage
###Linux
1.Compile & prepare files
  ```
  javac -d bin src/client/*.java
  javac -d bin src/server/*.java
  cp src/ssl* bin
  cd bin
  ```
2.Start server & client
  server:
  ```
  java server.MultiJabberServer
  ```
  client:
  ```
  java client.MultiJabberServer
  ```
